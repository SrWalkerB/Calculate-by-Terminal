# Calculate-by-Terminal

## Usage Dev:
### Sum

``
cargo run -- -o sum -v "10 15"
``

### Subtraction
``
cargo run -- -o subtraction -v "10 15"
``

### Multiplication
``
cargo run -- -o multiplication -v "10 15"
``

### Division
``
cargo run -- -o division -v "10 15"
``

## In terminal bash

``
cal -o sum -v "10 30 50"
``
